Smart Shopping using Bluetooth beacons

This application focuses on using Bluetooth beacons to provide indoor proximity information in a grocery store environment. The app assumes a grocery store which has several aisles on which different products are placed. Bluetooth Beacons will places on each of the store’s aisles. The app provides the user a context aware customized list of discounted products for the aisle closest to the user. Application is using Braintree payment API to process payments.
